# Online Shopping Cart Microservices

## Project Description
An online shopping website built with a microservices architecture. The application provides a scalable and modular approach for managing products and orders, supporting future enhancements like cart and order management.

## Features
- Product home page for users
- Admin functionalities:
  - Add a single product
  - Bulk product upload
  - Update or delete products
- Future features: Cart, order management, and more

## Tech Stack / Tools Used
- **Frontend:** Angular 20, Angular Material, HTML, CSS, JavaScript, TypeScript  
- **Backend:** Java 8, Spring Boot, Microservices architecture  
- **Authentication & Security:** Keycloak  
- **Databases:** MongoDB, PostgreSQL  
- **Caching:** Redis  
- **Containerization:** Docker
